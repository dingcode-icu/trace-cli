# tracer-cli
:raised_eyebrow:JUST MORE SIMPLE THEN WEB -  trace command line tool to view the traces;


### Desc 
rust 1.6+

main dependents:
* clap
* ureq
* prettytable-rs


### Build 
JUST 

>cargo build --release --target=$target 

### Run 
linux&&Mac:
> chmod a+x $path/trace_cli
> cd $path 
> ./trace_cli -h


windows:
>./trace_cli.exe -h
